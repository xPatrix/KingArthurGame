# KingArthurGame
This is a small game developed during meetings of student organization SKN Artefakty.

## Description
Player moves by clicking with left mouse button. Right mouse button is for attacking. Enemies are wandering around until player alerts them by attacking near them. Arthur is able to kill goblins as well as goblins are able to kill Arthur. GUI consists of icon of Arthur and red bar indicating health points located in the upper right corner.
